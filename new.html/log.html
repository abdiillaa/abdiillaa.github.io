Логарифм формулалары

Логарифм анықтамасы

\(\log_a b = c \iff a^c = b\)

Негізгі қасиеттері

\(\log_a 1 = 0\)

\(\log_a a = 1\)

\(\log_a a^x = x\)

\(a^{\log_a x} = x\)

Көбейту формуласы

\(\log_a (bc) = \log_a b + \log_a c\)

Бөлу формуласы

\(\log_a \frac{b}{c} = \log_a b - \log_a c\)

Дәреже формуласы

\(\log_a b^n = n\log_a b\)

Негізді ауыстыру формуласы

\(\log_a b = \frac{\log_c b}{\log_c a}\)

\(\log_a b = \frac{\ln b}{\ln a}\)

Арнайы логарифмдер

\(\ln x = \log_e x\)

\(\lg x = \log_{10} x\)

Қосымша формулалар

\(\log_a b = \frac{1}{\log_b a}\)

\(a^{\log_a b} = b\)

\(\log_a a^n = n\)

\(\log_a \sqrt[n]{b} = \frac{1}{n}\log_a b\)

\(\log_a (b^m c^n) = m\log_a b + n\log_a c\)

\(\log_a \frac{1}{b} = -\log_a b\)

Логарифм теңдеулеріне қолданылатын формулалар

\(\log_a f(x) = \log_a g(x) \Rightarrow f(x)=g(x)\)

\(\log_a f(x) = b \Rightarrow f(x)=a^b\)

\(a^{f(x)} = a^{g(x)} \Rightarrow f(x)=g(x)\)
