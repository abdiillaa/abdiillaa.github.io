Тригонометрия формулалары (100+)

Негізгі анықтамалар

\(\sin x = \frac{қарсы}{гипотенуза}\)

\(\cos x = \frac{жанасушы}{гипотенуза}\)

\(\tan x = \frac{\sin x}{\cos x}\)

\(\cot x = \frac{\cos x}{\sin x}\)

\(\sec x = \frac{1}{\cos x}\)

\(\csc x = \frac{1}{\sin x}\)

Негізгі тригонометриялық тождестволар

\(\sin^2 x + \cos^2 x = 1\)

\(1 + \tan^2 x = \sec^2 x\)

\(1 + \cot^2 x = \csc^2 x\)

Қосу формулалары

\(\sin(a+b)=\sin a\cos b+\cos a\sin b\)

\(\sin(a-b)=\sin a\cos b-\cos a\sin b\)

\(\cos(a+b)=\cos a\cos b-\sin a\sin b\)

\(\cos(a-b)=\cos a\cos b+\sin a\sin b\)

\(\tan(a+b)=\frac{\tan a+\tan b}{1-\tan a\tan b}\)

\(\tan(a-b)=\frac{\tan a-\tan b}{1+\tan a\tan b}\)

Қос бұрыш формулалары

\(\sin(2x)=2\sin x\cos x\)

\(\cos(2x)=\cos^2x-\sin^2x\)

\(\cos(2x)=2\cos^2x-1\)

\(\cos(2x)=1-2\sin^2x\)

\(\tan(2x)=\frac{2\tan x}{1-\tan^2 x}\)

Жарты бұрыш формулалары

\(\sin^2\frac{x}{2}=\frac{1-\cos x}{2}\)

\(\cos^2\frac{x}{2}=\frac{1+\cos x}{2}\)

\(\tan\frac{x}{2}=\frac{\sin x}{1+\cos x}\)

\(\tan\frac{x}{2}=\frac{1-\cos x}{\sin x}\)

Көбейтіндіден қосындыға

\(\sin a \sin b = \frac{1}{2}[\cos(a-b)-\cos(a+b)]\)

\(\cos a \cos b = \frac{1}{2}[\cos(a-b)+\cos(a+b)]\)

\(\sin a \cos b = \frac{1}{2}[\sin(a+b)+\sin(a-b)]\)

Қосындыдан көбейтіндіге

\(\sin a + \sin b = 2\sin\frac{a+b}{2}\cos\frac{a-b}{2}\)

\(\sin a - \sin b = 2\cos\frac{a+b}{2}\sin\frac{a-b}{2}\)

\(\cos a + \cos b = 2\cos\frac{a+b}{2}\cos\frac{a-b}{2}\)

\(\cos a - \cos b = -2\sin\frac{a+b}{2}\sin\frac{a-b}{2}\)
