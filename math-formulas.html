Математика формулалар

ҰБТ-да жиі кездесетін негізгі формулалар осы бетке топтастырылды. Бұл жинақ тез қайталауға арналған: алгебра, теңдеулер, тригонометрия, геометрия, прогрессия, комбинаторика және ықтималдық.

Алгебра

Негізгі түрлендірулер

Қысқаша көбейту формулалары

Квадрат қосындысы

(a + b)2 = a2 + 2ab + b2

Квадрат айырмасы

(a - b)2 = a2 - 2ab + b2

Айырма көбейтіндісі

(a + b)(a - b) = a2 - b2

Куб қосындысы

(a + b)3 = a3 + 3a2b + 3ab2 + b3

Куб айырмасы

(a - b)3 = a3 - 3a2b + 3ab2 - b3

Кубтар формуласы

Кубтар айырмасы

a3 - b3 = (a - b)(a2 + ab + b2)

Кубтар қосындысы

a3 + b3 = (a + b)(a2 - ab + b2)

Дәреже және түбір

Көбейту

am · an = am+n

Бөлу

am / an = am-n

Дәреженің дәрежесі

(am)n = amn

Көбейтіндінің дәрежесі

(ab)n = anbn

Теріс дәреже

a-n = 1 / an

Түбір

√ab = √a · √b, √(a / b) = √a / √b

Логарифм

Негізгі мәндер

loga1 = 0, logaa = 1

Көбейтінді

loga(bc) = logab + logac

Бөлінді

loga(b / c) = logab - logac

Дәреже

logabn = n logab

Негізді ауыстыру

logab = logcb / logca

Теңдеулер мен тізбектер

Квадрат, Виет, модуль

Квадрат теңдеу

Жалпы түрі

ax2 + bx + c = 0, a ≠ 0

Дискриминант

D = b2 - 4ac

Түбірлер

x1,2 = (-b ± √D) / 2a

Толымсыз түрлер

ax2 + bx = 0 → x(ax + b) = 0

Виет теоремасы

Түбірлер қосындысы

x1 + x2 = -b / a

Түбірлер көбейтіндісі

x1x2 = c / a

Модуль

Анықтама

|x| = x, x ≥ 0; |x| = -x, x < 0

Негізгі қасиет

|ab| = |a|·|b|, |a / b| = |a| / |b|

Арақашықтық

|x - a| = d

Функция және график

Түзу сызық

y = kx + b

Парабола төбесі

x0 = -b / 2a, y0 = -D / 4a

Қашықтық формуласы

d = √((x2 - x1)2 + (y2 - y1)2)

Тригонометрия

sin, cos, tg

Негізгі теңдіктер

Пифагорлық теңдік

sin2x + cos2x = 1

Тангенс

tg x = sin x / cos x

Котангенс

ctg x = cos x / sin x

Қосымша теңдіктер

1 + tg2x = 1 / cos2x

Қосымша теңдіктер

1 + ctg2x = 1 / sin2x

Қосу және айыру

Синус

sin(α ± β) = sin α cos β ± cos α sin β

Косинус

cos(α ± β) = cos α cos β ∓ sin α sin β

Тангенс

tg(α ± β) = (tg α ± tg β) / (1 ∓ tg α tg β)

Қос бұрыш және жарты бұрыш

Синус

sin 2x = 2 sin x cos x

Косинус

cos 2x = cos2x - sin2x

Балама түрлері

cos 2x = 2cos2x - 1 = 1 - 2sin2x

Жарты бұрыш

sin2x = (1 - cos 2x) / 2

Жарты бұрыш

cos2x = (1 + cos 2x) / 2

Негізгі мәндер

30°

sin 30° = 1 / 2, cos 30° = √3 / 2, tg 30° = √3 / 3

45°

sin 45° = cos 45° = √2 / 2, tg 45° = 1

60°

sin 60° = √3 / 2, cos 60° = 1 / 2, tg 60° = √3

Радиан

180° = π рад, 90° = π / 2

Жазықтық геометриясы

Үшбұрыш, шеңбер, көпбұрыш

Үшбұрыш

Периметр

P = a + b + c

Жарты периметр

p = (a + b + c) / 2

Аудан

S = 1 / 2 · a · h

Екі қабырға және бұрыш

S = 1 / 2 · ab sin C

Герон формуласы

S = √(p(p - a)(p - b)(p - c))

Пифагор теоремасы

c2 = a2 + b2

Үшбұрыш теоремалары

Синустар теоремасы

a / sin A = b / sin B = c / sin C = 2R

Косинустар теоремасы

c2 = a2 + b2 - 2ab cos C

Төртбұрыштар

Тік төртбұрыш

S = ab, P = 2(a + b)

Шаршы

S = a2, P = 4a, d = a√2

Параллелограмм

S = a · h = ab sin α

Ромб

S = 1 / 2 · d1d2

Трапеция

S = (a + b)h / 2

Шеңбер

Ұзындығы

L = 2πr = πd

Ауданы

S = πr2

Доға ұзындығы

l = πrα / 180°

Сектор ауданы

S = πr2α / 360°

Стереометрия

Көлем мен толық бет

Призма және параллелепипед

Призма көлемі

V = Sтабан · h

Тікбұрышты параллелепипед

V = abc

Толық бет

S = 2(ab + bc + ac)

Диагональ

d = √(a2 + b2 + c2)

Цилиндр және конус

Цилиндр көлемі

V = πr2h

Цилиндр толық беті

S = 2πr(r + h)

Конус көлемі

V = 1 / 3 · πr2h

Конус толық беті

S = πr(r + l)

Шар

Көлемі

V = 4 / 3 · πr3

Бет ауданы

S = 4πr2

Пирамида

Көлемі

V = 1 / 3 · Sтабан · h

Бүйір беті

Sбүйір = 1 / 2 · Pтабан · l

Прогрессиялар

Арифметикалық және геометриялық

Арифметикалық прогрессия

n-ші мүшесі

an = a1 + (n - 1)d

Айырмасы

d = an+1 - an

Қосындысы

Sn = (a1 + an)n / 2

Балама түрі

Sn = (2a1 + (n - 1)d)n / 2

Геометриялық прогрессия

n-ші мүшесі

bn = b1qn-1

Еселігі

q = bn+1 / bn

Қосындысы

Sn = b1(qn - 1) / (q - 1)

Шексіз кемімелі

S = b1 / (1 - q), |q| < 1

Комбинаторика және ықтималдық

Санау формулалары

Факториал және орналастыру

Факториал

n! = 1 · 2 · 3 · ... · n

Алмастыру

Pn = n!

Орналастыру

Ank = n! / (n - k)!

Теріп алу

Cnk = n! / (k!(n - k)!)

Ықтималдық

Классикалық формула

P(A) = m / n

Қарама-қарсы оқиға

P(Ā) = 1 - P(A)

Қосу формуласы

P(A + B) = P(A) + P(B) - P(AB)

Көбейту формуласы

P(AB) = P(A) · P(B), тәуелсіз болса

Тез есте сақтау

Квадрат теңдеуде алдымен D табыңыз. Үшбұрыш есептерінде аудан үшін көбіне 1 / 2 · ah, ал бұрыш берілсе 1 / 2 · ab sin C формуласын тексеріңіз.

Тригонометрия кеңесі

Көп есепте sin2x + cos2x = 1 және sin 2x = 2 sin x cos x формулалары негізгі қысқарту құралы болады.

Прогрессия кеңесі

Айырмасы бірдей болса арифметикалық, еселігі бірдей болса геометриялық прогрессия. Есепті бастамас бұрын осыны ажыратып алыңыз.

Бұл бет тез қайталау үшін жасалды. Қаласаңыз, келесі қадамда осы бетке формулаларды суретпен, тақырыптық фильтрмен немесе іздеумен де кеңейтіп бере аламын.
