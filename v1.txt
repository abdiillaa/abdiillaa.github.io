Жоба
  Бұл GitHub Pages-ке арналған static сайт, негізгі мақсаты ҰБТ дайындық. Негізгі стек: index.html + style.css + test/app.js + test/
  style.css. Backend жоқ. Telegram-ға login/result хабарламалары қазір тікелей frontend-тен жіберіледі, яғни бұл әдейі insecure,
  уақытша шешім.

  Негізгі файлдар

  - index.html (/home/linx/zarevenge.github.io/index.html): login screen, main layout, sidebar/menu, home/profile/test/settings/
    ranking/materials беттері, manifest/favicon link-тері.
  - style.css (/home/linx/zarevenge.github.io/style.css): негізгі сайттың стилі.
  - test/app.js (/home/linx/zarevenge.github.io/test/app.js): бүкіл логика. Login, user/avatar, Telegram, test runner, modes, result/
    history, widgets, mobile runner UI.
  - test/style.css (/home/linx/zarevenge.github.io/test/style.css): test runner стилі.
  - manifest.json (/home/linx/zarevenge.github.io/manifest.json): PWA manifest.
  - Иконкалар: 1.png (/home/linx/zarevenge.github.io/1.png), 2.png (/home/linx/zarevenge.github.io/2.png), 3.png (/home/linx/
    zarevenge.github.io/3.png), 4.png (/home/linx/zarevenge.github.io/4.png).
  - Reference images: mendeleev.jpg, erigish.png.

  Қазір жұмыс істейтін негізгі мүмкіндіктер

  - Login local encrypted user жүйесімен істейді.
  - Main pages: Басты бет, Профиль, Тест, Баптаулар, Рейтинг, Оқу материалдары, Шығу.
  - Test modes:
      - exam = Еркін режим және әдепкі режим.
      - instant = бірден тексеретін режим.
      - self-check = түртіп оқу режимі.
  - Question shuffle қайта қосылған, сұрақтар random.
  - Test runner mobile-ге бейімделген.
  - Runner header-де avatar + name бар.
  - Runner-де sidebar tools:
      - калькулятор
      - менделеев
      - ерігіштік
      - аяқтау
      - артқа
  - Mobile-де tools Menu арқылы ашылады.
  - Finish modal қазір толық кесте емес, бір жол summary.
  - Result history currentUser.testHistory ішінде сақталады.
  - Reset localStorage-ты толық тазартады.
  - Шығу logout қылады, кейін қайта кіргенде home автоматты ашылады.
  - PWA manifest және favicon қосылған.

  Telegram интеграциясы

  - Токен мен chat_id ашық түрде test/app.js (/home/linx/zarevenge.github.io/test/app.js)-те тұр.
  - Login event Telegram-ға жібереді:
      - Оқушы
      - IP
      - Құрылғы
      - Уақыты
  - Test result Telegram-ға жібереді:
      - Оқушы
      - Пән
      - Тест
      - Режим
      - Тапсырған уақыты
      - Уақыт
      - Нәтиже
      - Пайыз
      - Қате
  - Бұл шешім қауіпсіз емес, бірақ пайдаланушының өз сұрауымен солай қалдырылған.

  UI/UX бойынша маңызды соңғы шешімдер

  - Соңғы жасалған үлкен visual polish толық rollback жасалды.
  - Сондықтан style.css қазір бұрынғы, қарапайым нұсқада.
  - About Us блогы қосылып, кейін толық алынып тасталды.
  - User minimalist бағытты қалайды, артық feature қоспау керек.
  - User-ге көбіне “clean, simple, beautiful” керек, бірақ overly redesigned нұсқалар ұнамай қалуы мүмкін.

  Mobile runner туралы маңызды детальдар

  - Алдыңғы/Келесі/Аяқтау navigation phone-да fixed/floating логикамен біраз өзгертілген, қазір қолдануға болатын нұсқада.
  - Menu батырмасы header-де тұр, үш сызық icon.
  - Menu modal overlay progress bar мен navigation үстін жабатындай түзетілген.
  - Option таңдағанда page jump/scroll мәселесі бұрын түзетілген.
  - Алдыңғы пән/Келесі пән runner-ден алынып тасталған.

  iPhone/login туралы

  - Login input-та autocapitalize, autocorrect, spellcheck өшірілген.
  - Password тексеру .normalize('NFC') арқылы жүреді.
  - iPhone-да http://LAN-IP кезінде crypto.subtle secure context мәселесі болуы мүмкін. GitHub Pages https-те жақсырақ істейді.

  PWA/icon күйі

  - Manifest жаңартылған: ҰБТ дайындық.
  - index.html-де manifest, theme-color, apple-mobile-web-app-*, apple-touch-icon, favicon link-тері бар.
  - Негізгі icon ретінде 1.png қолданылып тұр.

  Пайдаланушы талғамы

  - Backend, analytics, үлкен feature set міндетті емес деп санайды.
  - Сайт 3 шақты адамға ғана арналуы мүмкін, бұл қалыпты деп қабылдайды.
  - Минимализм мен practical UX-ті қалайды.
  - Артық блоктар ұнамаса, тез алып тастауды сұрайды.
  - Көп жағдайда шағын, нақты patch керек.
