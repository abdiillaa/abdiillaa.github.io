# ҰБТ дайындық

Бұл репо — GitHub Pages‑те жұмыс істейтін статикалық оқу платформасы. Мақсаты: ҰБТ‑ға дайындықты жеңілдету үшін тесттер, материалдар және оқу кестесін бір жерде ұстау.

**Мүмкіндіктер**
- ПИН‑кодпен кіру экраны
- Апталық оқу кестесі (Home)
- Пәндер бойынша тест тапсыру (Қазақстан тарихы, Информатика)
- Тесттегі сұрақ санын баптау (Settings)
- Сынып рейтингі бөлімі (Academic Ranking)
- Study Materials: серверсіз JSON‑негізді материалдар

**Жылдам бастау**
1. Браузерде `index.html` файлын ашыңыз.
2. Немесе локал сервер көтеріңіз:

```bash
python -m http.server 8000
```

Сосын `http://localhost:8000` ашыңыз.

**Жоба құрылымы**
- `index.html` — негізгі бет, навигация және бөлімдер.
- `style.css` — жалпы стильдер.
- `test/app.js` — тест логикасы, сұрақ таңдау, нәтижені есептеу.
- `test/data/*.json` — тест сұрақтары.
- `materials-data/` — Study Materials деректері.
- `manifest.json` — PWA метадеректері.

**Тест қосу**
1. `test/data/` ішіне жаңа JSON файл қосыңыз.
2. `test/app.js` ішінде `files` объектісіне ID және файл жолын қосыңыз.
3. `index.html` ішінде тест тізіміне жаңа жол (`testgo(ID)`) қосыңыз.

**Тест дерек форматы**
```json
[
  {
    "question": "Сұрақ мәтіні?",
    "options": ["A", "B", "C", "D"],
    "answer": "B"
  }
]
```

Ескерту: дұрыс жауапты анықтау логикасы:
- `correct` болса — соны алады
- әйтпесе `answer` болса — соны алады
- екеуі де жоқ болса — `options[0]` дұрыс деп саналады

**Материалдар дерек форматы**
Материалдар `materials-data/README.md` ішінде толық сипатталған. Қысқаша:
- `materials-data/catalog.json` — пәндер мен тақырыптар тізімі
- `materials-data/topics/*.json` — `{question, answer}` жұптары

**Ұсыныс**
Тесттерді интернеттен көшіргенде, жауап кілтін міндетті түрде тексеріңіз. Жауап қателігі барлық нәтиже сапасына әсер етеді.
